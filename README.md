A 3D game made in GoDot. 
A first person shooter game - maneuver your way through a custom level, kill robots with different weapons, pick up ammo and dodge lava while getting to the finish. 
This project was made following gamedev.tv tutorials. 
To run the game, download repository and open in GoDot. 
Alternatively, you can download games I've created following this link: https://jeffbenzos.itch.io/
